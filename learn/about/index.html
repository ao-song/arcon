Overview

Arcon is a runtime for writing real-time analytics applications with the Rust programming language. At the core, Arcon aims to enable users to build scalable stateful streaming applications with event-time semantics.

Highlights

Rust

Arcon is written entirely in Rust for performance, memory safety, and no garbage collection.

Apache Arrow

Arcon has first-class integration with the Apache Arrow™ format.

Middleware

Arcon utilises
Kompact, a state-of-the-art
message passing framework.

Temporal Stream State
